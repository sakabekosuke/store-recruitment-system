# データフロー仕様

店舗採用システム Version 1.0.0

---

# このドキュメントについて

本ドキュメントは「店舗採用システム」におけるデータの流れ、自動処理、各シート間の関係を定義する仕様書です。

システムの改善・保守・機能追加を行う際は、本仕様書を基準として設計してください。

Google Apps ScriptやGoogleスプレッドシート関数を変更する場合は、必ず本データフローへの影響を確認してください。

---

# システム全体フロー

```text
応募者
   │
   ▼
Googleフォーム
   │
   ▼
応募データ
   │
   │（onFormSubmit）
   ▼
採用管理
   │
   ▼
店長・採用担当者
   │
   ├───────────────┐
   │               │
   ▼               ▼
面接日時入力      ステータス変更
   │               │
   └──────┬────────┘
          │
          ▼
ダッシュボード更新
          │
          ▼
採用・不採用・辞退
          │
          │（onEdit）
          ▼
応募データへ採用結果保存
          │
          ▼
採用管理から自動削除
          │
          ▼
ダッシュボード更新
```

# 保守ルール

新しい機能を追加する場合は

必ず本データフローへ追記してください。

Google Apps Scriptを変更する場合は

データの流れが変わらないことを確認してください。

データベースである「応募データ」を直接編集する処理は追加しないでください。

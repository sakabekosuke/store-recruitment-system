# 📦 店舗採用システム Template

Googleフォーム・Googleスプレッドシート・Google Apps Script を利用して構築する
**店舗向け採用管理システム テンプレート**です。

本テンプレートを利用することで、応募受付から採用管理までを約30分で構築できます。

---

## 🎯 このテンプレートでできること

- Googleフォームで応募受付
- 応募情報を自動でスプレッドシートへ登録
- 採用状況を一元管理
- ダッシュボードで今日対応すべき応募を可視化
- 採用・不採用履歴の保存
- 他店舗への横展開

---

# 🚀 導入手順

テンプレートの導入は以下の順番で行います。

1. Googleフォーム作成
2. Googleスプレッドシート作成
3. Apps Script設定
4. 初期設定
5. 動作確認

詳細は docs フォルダを参照してください。

---

# 📁 フォルダ構成

```
template/
```

# 📚 ドキュメント

|ファイル|内容|

# ⚙️ 必要なもの

- Googleアカウント
- Googleフォーム
- Googleスプレッドシート

---

# 💡 カスタマイズ

以下の項目は店舗ごとに自由に変更できます。

- シート名
- ステータス名
- 面接日管理
- ダッシュボード表示
- 列構成

変更方法は Apps Script の `00_config.gs` にまとめています。

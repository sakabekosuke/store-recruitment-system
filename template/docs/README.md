# Documents

店舗採用システム Template の導入マニュアルです。

---

## 📖 読む順番

1. 01_初期設定
2. 02_テンプレートをコピー
3. 03_Apps Script設定
4. 04_導入チェック

この順番で進めることで、
約15〜20分で導入できます。

---

## 📄 ファイル一覧

|ファイル|内容|
|---|---|
|01_初期設定.md|導入前の準備・必要なものを確認します。|
|02_テンプレートをコピー.md|Googleフォーム・Googleスプレッドシートをコピーし、連携します。|
|03_Apps Script設定.md|店舗番号・通知先メールアドレス・トリガーを設定します。|
|04_導入チェック.md|動作確認を行い、導入完了を確認します。|

# 📋 02_Googleフォーム作成

🟩🟩⬜⬜⬜　**進捗：STEP 2 / 5**

---

# はじめに

この章では、
店舗採用システムで使用するGoogleフォームを準備します。

Template版では、
マスターフォームをコピーして利用します。

**所要時間：約3分**

---

# この章で行うこと

- Googleフォームをコピーする
- フォーム名を変更する
- 回答先スプレッドシートを設定する
- 動作確認を行う

---

# Googleフォームをコピー

Google Drive上の

**master_アルバイト応募フォーム**

を開きます。

↓

**ファイル**

↓

**コピーを作成**

を選択してください。

コピー後は、
店舗名など分かりやすい名前へ変更してください。

例）

```
アルバイト応募フォーム（〇〇店）
```

---

# フォーム内容について

Template版には、
応募に必要な質問があらかじめ設定されています。

質問内容や回答形式については、

**Googleフォーム設計書.md**

をご確認ください。

---

# 回答先を設定

Googleフォーム上部

**回答**

↓

**スプレッドシートアイコン**

↓

**既存のスプレッドシートを選択**

をクリックします。

↓

本システム用にコピーした

**Googleスプレッドシート**

を選択してください。

---

# シート名を変更

Googleフォームを連携すると、

```
フォームの回答 1
```

（または Form Responses 1）

というシートが自動作成されます。

シート名を

```
応募管理
```

へ変更してください。

Apps Scriptは
このシート名を参照します。

---

# 動作確認

テストとして1件応募してください。

以下を確認します。

- 応募管理シートへ保存される
- 回答内容が正しく保存される

※ この時点では採用管理シートへは登録されません。

Apps Script設定後に自動登録されます。

---

# よくある間違い

### 新しいスプレッドシートを作成してしまった

必ず

**コピーしたスプレッドシート**

を回答先に設定してください。

---

### シート名を変更していない

```
応募管理
```

へ変更してください。

Apps Scriptはこのシート名を利用します。

---

### フォームの質問を変更してしまった

質問の追加・削除・順番の変更を行う場合は、

Googleフォーム設計書と
Apps Scriptの設定も合わせて変更してください。

# Googleスプレッドシート対応

フォームの回答は、
Googleフォームと連携したスプレッドシートへ自動保存されます。

導入後は、

```
フォームの回答 1
```

（または Form Responses 1）

を

```
応募管理
```

へ変更してください。

Apps Scriptは

**「応募管理」**

というシート名を参照します。

# 運用ルール

このフォームは
**マスターフォーム**です。

各店舗で利用する場合は、

```
ファイル
　↓
コピーを作成
```

を行い、

コピーしたフォームを利用してください。

マスターフォームは直接編集しません。

# 導入の流れ

① Googleフォームをコピー

↓

② Googleスプレッドシートをコピー

↓

③ Googleフォームの回答先を
Googleスプレッドシートへリンク

↓

④ 自動作成された

```
フォームの回答 1
```

を

```
応募管理
```

へ変更

↓

⑤ 設定シートで店舗番号を選択

↓

⑥ 利用開始
